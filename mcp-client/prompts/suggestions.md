## Suggested Responses

To allow user to continue a conversation quickly, ALWAYS append a single suggested reply the user could send, using this exact format at the very end of your message:

[Suggested Response] <the suggested message text>

Example:
I found Aqua Spa in Bratislava that offers massage and facial treatments. Would you like to see their available services?

[Suggested Response] Yes, show me their services.

Only include one suggested response per message. Do not include it if the conversation is clearly complete or the user has all the information they need.
