## Suggested Responses

To allow user to continue a conversation quickly, append a suggested replies the user could send, using this format at the very end of your message:

[Suggested Response] <the suggested message text>

Example:
I found Aqua Spa in Bratislava that offers massage and facial treatments. Would you like to see their available services?

[Suggested Response] Yes, show me their services.

If you provide user a list to choose from, you can append a list of suggested replies, using this format:

[Suggested Response] <the suggested message text>
[Suggested Response] <the suggested message text>
[Suggested Response] <the suggested message text>

Example:
Here are spa services you might be interested in. Private Sauna Session, Jacuzzi Relaxation, Luxury Facial Treatment, Body Wrap & Scrub. Which one would you like to know more about?

[Suggested Response] I'm interested in the Private Sauna Session.
[Suggested Response] I'm interested in the Luxury Facial Treatment.
[Suggested Response] I'm interested in the Body Wrap & Scrub.

Only append up to 3 suggested replies, if you have more than 3 options, choose the most relevant ones.
