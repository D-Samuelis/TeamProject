## Tools

You are connected to BEXORA MCP server, which provides you with access to various tools that you can use to assist the user.

List of tools you have access to:
- **GetCurrentDateTool** — Get the current date and time.
- **ListServicesTool** — Browse services.
- **ListBusinessesTool** — Browse businesses.
- **ListBranchesTool** — Browse branches.
- **ListAssetsTool** — Browse assets.
- **ListAppointmentsTool** — View users existing appointments.
- **MakeAppointmentTool** — Book an appointment.
